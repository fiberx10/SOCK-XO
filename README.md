# SOCK-XO
online xo game based on <strong> PYTHON </strong> can be played online (internet) or localy  .
### TECHNOLOGIES USED
basically the game is built using python and python socket for network 

### HOW TO RUN

To run this game , you should first run the server  : 
 > python server.py
 
Then connect your computers to the same network 
open cmd/terminal on all of them and run
> pyhton client.py
